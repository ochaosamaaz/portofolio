# Assets

Place the following files here:

| Filename | Description |
|----------|-------------|
| `profile.jpg` | Your profile photo (square, min 200x200px) |
| `CV-Ossama-Bintang.pdf` | Your CV/Resume PDF for download |
| `og-image.png` | Social sharing preview image (1200x630px recommended) |

## Tips:
- Profile photo: crop to square, keep under 200KB
- OG image: use a screenshot of the hero section or create a branded banner
